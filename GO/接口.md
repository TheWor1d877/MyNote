空接口是所有类型的超集 —— 因为它不包含任何方法，所以任何类型都自动满足它。
Go 的 空接口 interface{}（Go 1.18+ 后等价于 any）是带类型信息的安全容器。
## 类型转换与类型断言
特性	类型转换	类型断言
适用对象	具体类型之间	接口类型
语法	T(v)	v.(T)
失败时	编译错误	返回 false 或 panic
底层检查	编译时检查	运行时检查

## 类型 switch：多类型处理
```go
func describe(x interface{}) {
    switch v := x.(type) {
    case int:
        fmt.Println("Integer:", v)
    case string:
        fmt.Println("String:", v)
    case bool:
        fmt.Println("Boolean:", v)
    case Dog:
        fmt.Println("Dog named:", v.Name)
    default:
        fmt.Println("Unknown type")
    }
}

// 使用
describe(42)        // Integer: 42
describe("hi")      // String: hi
describe(Dog{"Max"})// Dog named: Max
```
