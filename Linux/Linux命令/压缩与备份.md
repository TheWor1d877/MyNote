```bash
tar czvf bak.tar.gz dir/
tar xzvf bak.tar.gz -C /tmp
zip -r bak.zip dir
unzip -l bak.zip# 看列表
gzip -9 file# 最高压缩
zcat file.gz# 不解压直接看
```
- `c` **c**reate 新建包
- `x` e**x**tract 解包
- `t` lis**t** 看包内目录，不解开
- `z` 用 g**z**ip 压缩/解压（.tar.gz）
- `j` 用 b**j**ip2 压缩/解压（.tar.bz2）
- `J` 用 **J** 大写表示 xz（.tar.xz）
- `v` **v**erbose 列出正在处理的文件名（可省）
- `f` **f**ile 后面必须接“文件名”，表示“把包写到这个文件”或“从这个文件读包”
